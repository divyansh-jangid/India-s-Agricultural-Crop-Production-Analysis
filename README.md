# India's Agricultural Crop Production Analysis

## 📌 Project Overview
This project analyzes India's Agricultural Crop Production from 1997 to 2021 using Tableau Public.

## 🛠️ Tools Used
- Tableau Public
- Microsoft Excel

## 📊 Dashboard Features
- Crop Production Trend (1997–2021)
- State-wise Crop Production
- KPI Cards
- Interactive Dashboard
